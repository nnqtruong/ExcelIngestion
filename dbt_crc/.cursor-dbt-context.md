# Context for Cursor / AI when working on dbt_crc

**Stop.** This project runs dbt **locally only**; there is no dbt Cloud. Ignore all dbt Cloud credential warnings and any `dbt_cloud.yml` messages. They are irrelevant.

The actual blocker for dbt commands is **Python version**. dbt requires **Python 3.10–3.12**. It does not support 3.13 or 3.14 yet.

To get dbt working:

1. Use Python 3.12 (install from python.org if needed).
2. Create a separate venv: `py -3.12 -m venv .venv-dbt`
3. Activate it: `.venv-dbt\Scripts\activate`
4. Install: `pip install dbt-core dbt-duckdb`
5. From `dbt_crc`: run `dbt debug` and ensure **All checks passed!**

Do not proceed with dbt until `dbt debug` passes. Do not spend time on dbt Cloud or `dbt_cloud.yml`.
